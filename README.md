# RPA-Admin

## Component Docs

- [AutoRoles Cog](docs/autoroles-cog.md)
- AutoInvite Cog: configure `JSON/serverconfig.json` with an `auto_invite` block so members who receive mapped roles in the main server are DM'd a unique one-use invite to the configured destination server. These invites stay valid until redeemed, then immediately expire because they are single-use.
